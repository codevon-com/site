# Team Images

Place team member images in this directory with the following filenames:

- `bueno.jpg` - Bueno, Founder
- `barbosa.jpg` - Barbosa, Founder  
- `meyer.jpg` - Meyer, Founder
- `alex.jpg` - Alex, Engineer
- `victor.jpg` - Victor, Engineer
- `kalleo.jpg` - Kalleo, Engineer

**Note:** Images will be displayed in grayscale on the site. 
Recommended: Square aspect ratio (e.g., 800x800px) for best results.
